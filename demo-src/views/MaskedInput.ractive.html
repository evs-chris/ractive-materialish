This decorator takes an input field and applies a mask that accepts numbers, letters, or letters and numbers at each mask position. Cursor movement is automatically handled, and the field is filled with the mask, so no weird tricks are needed to display a prompt for the user. ### Plugin options All options are optional. * `name: string ='masked'` - the name to use when registering the decorator as a plugin ### Arguments This decorator takes up to two strings and an options has as arguments. The first and second may be strings that correspond to the `mask` and `value` options. The first, second, or third may be an options hash. * `mask: string` - the mask to use in the input * #### Characters `#` - matches any number character `?` - matches any word character `@` - matches any upper or lowercase letter a-z Any other characters are taken literally and automatically included in the masked value. * `value: keypath` - a keypath to use to bind the input value * `display: keypath` - a keypath to use to bind the displayed value * `lazy: boolean = true` - whether or not to immediately update the bound values Template: ```hbs ``` Result:
