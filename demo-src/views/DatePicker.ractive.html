The `DatePicker` displays a calendar that allows selecting a date. ### Plugin Options All options are optional. * `name: string = 'date-picker'` - the name to use when registering the component ### Attributes All attributes are optional. * `date: binding` - the value to which the date should be bound * `mode: 'month'|'months'|'year' = 'month'` - the mode for the picker ### Template: ```handlebars ``` ### Result: 
      Date: {{date}}
