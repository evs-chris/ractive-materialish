This is your standard boolean toggle that looks like an on/off switch. This is actually a macro partial rather than a component, so it's fairly light weight. It also supports `null` or `undefined` states with an intermediate position for all of your parties with three-state booleans. ### Plugin options All options are optional. * `name: string = 'toggle'` - the name to use when registering the plugin as a component ### Attributes * `disabled: boolean` - whether the toggle is disabled * `value: any` - the backing value for the toggle * `nullable: boolean` - whether the toggle includes a `null` state * `undefined: boolean` - whether the null value should be `undefined` rather than `null` ### Template: ```hbs {{'' + .foo}} ``` ### Result: {{'' + .foo}}
